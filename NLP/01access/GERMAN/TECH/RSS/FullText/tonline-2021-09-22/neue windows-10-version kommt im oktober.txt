t-online.de, avr 
Ein Rechner mit Windows 10 (Symbolbild): Im Oktober soll die neue Version von Windows 10 erscheinen.  (Quelle:  Robert Günther/dpa)Für Windows 10 erscheinen im Jahr zwei große Updates: Im Oktober soll nun das große Herbst-Update kommen. Wir zeigen, was Nutzer erwarten können. So sieht das neue Windows 11 ausIm Oktober wird das nächste Funktions-Update für Windows 10 erscheinen: Windows 10 Version 21H2. Nutzer von kommerziellen Rechnern können zum Testen bereits auf das Update zugreifen, wie verschiedene Fachseiten berichten. Privatnutzer erhalten das Update offiziell ab Oktober. Im Vergleich zu früheren Aktualisierungen gibt es nur kleinere Änderungen. Wir zeigen, was Sie erwartet. Das ändert sich in Windows 10 Version 21H2  Das neue Funktions-Update bietet im Grunde drei Neuerungen, wie Microsoft auf seiner Website berichtet. So soll das Betriebssystem die Wi-Fi-Sicherheit verbessern. Dafür wird Windows 10 den WPA3 H2E-Standard unterstützen. WPA3 ist der neue WLAN-Standard, mehr dazu lesen Sie hier. H2E steht für Hash-to-Element und ist eine Variante von WPA3, die das Netzwerk gegen bestimmte Seitenkanalangriffe absichern soll. Wer das Windows Subsystem for Linux (WSL) nutzt, wird ebenfalls vom kommenden Oktober-Update profitieren. Denn laut Microsoft wird es für solche Systeme sowie Azure IoT Edge for Linux on Windows (EFLOW) eine GPU-Unterstützung geben. Das heißt, dass das System auf die Grafikkarte für rechenintensive Aufgaben zugreifen kann. Microsoft nennt ein Beispiel maschinelles Lernen. Als dritte neue Funktion nennt Microsoft "vereinfachte kennwortlose Deployment-Modelle" für Windows Hello for Business. Mit Windows Hello for Business können sich Nutzer – ähnlich wie bei Windows Hello – an ihrem Rechner ohne Passwort anmelden. Stattdessen werden beispielsweise biometrische Daten genutzt. Durch das kommende Update soll es schneller möglich sein, Windows Hello for Business auf mehreren Rechnern einzurichten.Auch Windows 11 erscheint im Oktober Auch das neue Windows 11 erscheint im Oktober: Am 5. Oktober will Microsoft das Betriebssystem freigeben. Ursprünglich hieß es, dass Windows 11 Ende des Jahres erscheinen soll. Mehr dazu lesen Sie hier. Alle relevanten Infos zu Windows 11 wie Neuerungen oder Kosten finden Sie in dieser Übersicht. Windows 11 ist für Besitzer von Windows 10 kostenlos. Allerdings wird es nicht für alle Rechner verfügbar sein. Um das Betriebssystem installieren zu können, muss der Computer bestimmte Voraussetzungen erfüllen, etwa einen bestimmten Chipsatz und Prozessor nutzen.
